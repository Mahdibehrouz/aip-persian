# آلوتار - فروشگاه محصولات ارگانیک و طبیعی

یک فروشگاه آنلاین کامل و مدرن برای فروش محصولات ارگانیک و طبیعی با پشتیبانی کامل از زبان فارسی و RTL.

## 🌿 ویژگی‌های کلیدی

### طراحی و رابط کاربری
- **طراحی RTL کامل** با پشتیبانی از زبان فارسی
- **رنگ‌بندی سبز، سفید و قرمز** مطابق با درخواست
- **انیمیشن‌ها و افکت‌های زیبا** برای تجربه کاربری بهتر
- **طراحی ریسپانسیو** برای تمام دستگاه‌ها
- **فونت Vazirmatn** برای بهترین نمایش متن فارسی

### صفحات اصلی
1. **صفحه اصلی (index.html)**
   - هدر کامل با جستجو و منوی ناوبری
   - اسلایدر قهرمان با 3 اسلاید
   - بخش ویژگی‌ها (۱۰۰٪ طبیعی، ارسال سریع، کیفیت تضمینی)
   - دسته‌بندی محصولات
   - کاروسل محصولات ویژه
   - فرم عضویت در خبرنامه
   - فوتر کامل

2. **صفحه سبد خرید (cart.html)**
   - نمایش محصولات اضافه شده
   - امکان تغییر تعداد و حذف محصولات
   - محاسبه هزینه‌ها (محصولات، ارسال، تخفیف)
   - پیوند به صفحه پرداخت

3. **صفحه درباره ما (about.html)**
   - داستان شرکت و تاریخچه
   - ارزش‌ها و اهداف
   - آمار و اعداد مهم
   - ماموریت و چشم‌انداز

4. **صفحه پرداخت (payment.html)**
   - فرم اطلاعات شخصی
   - آدرس تحویل
   - انتخاب روش ارسال
   - انتخاب روش پرداخت
   - خلاصه سفارش و کد تخفیف

### قابلیت‌های فنی

#### JavaScript (script.js)
- **اسلایدر خودکار** با ناوبری دستی
- **سیستم سبد خرید کامل** با LocalStorage
- **سیستم علاقه‌مندی‌ها (Wishlist)**
- **کاروسل محصولات** با ناوبری
- **جستجوی لحظه‌ای**
- **سیستم اعلانات (Notifications)**
- **عضویت در خبرنامه**

#### CSS (style.css)
- **متغیرهای CSS** برای مدیریت آسان رنگ‌ها
- **Flexbox و Grid** برای چیدمان
- **انیمیشن‌ها و Transitions**
- **طراحی ریسپانسیو** با Media Queries
- **افکت‌های Hover** برای محصولات
- **سایه‌ها و گرادیانت‌ها**

### دسته‌بندی محصولات
- گیاهان دارویی
- روغن‌های طبیعی
- مراقبت از پوست
- مکمل‌های غذایی
- چای‌های گیاهی

### ویژگی‌های تجاری
- **ارسال رایگان** برای سفارشات بالای ۵۰۰ هزار تومان
- **پشتیبانی ۲۴/۷**
- **ضمانت کیفیت و اصالت**
- **امکان بازگشت کالا**
- **پرداخت آنلاین و در محل**

## 🚀 نحوه اجرا

### پیش‌نیازها
- مرورگر مدرن (Chrome, Firefox, Safari, Edge)
- سرور وب محلی (اختیاری)

### اجرای محلی
```bash
# استفاده از Python HTTP Server
python3 -m http.server 8000

# یا استفاده از Node.js
npx serve .

# یا استفاده از PHP
php -S localhost:8000
```

سپس به آدرس `http://localhost:8000` مراجعه کنید.
